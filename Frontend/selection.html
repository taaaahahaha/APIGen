These are just general instructions to export your sql query logs and you can use them as a guide
  • Enable query logging by setting the log_output parameter to TABLE in the my.cnf file. Use the command : “[mysqld] log = /path/to/query.log”
  • Alternatively, you can use the --log-output option when starting the mysql command-line client to specify the location of the query log file.
  • Use the SELECT and INTO OUTFILE statements to export the query logs to a file
  • For example, the following query will export all of the query logs to a file named query_logs.txt in the /tmp directory: "SELECT * FROM mysql.general_log INTO OUTFILE '/tmp/query_logs.txt';"
